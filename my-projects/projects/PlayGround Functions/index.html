PlayGrounds Functions

Esses projetos utilizam essa interface gráfica para experimento das funções, o objetivo não foi estilizar com esmero a página. O objetivo é demosntrar as funções com suas funcionalidades.

Comparar dois elementos

Essa função pegará duas palavras e retornará se elas são iguais

Separa strings

Essa função pegará uma frase/tTexto e irá separar as palavras por vírgula

Concatenar Strings

Recebe uma string e deve retornar uma string com o formato `'ÚLTIMA PALAVRA, PRIMEIRA PALAVRA`, independente do tamanho do array.

Contar Pontos

Recebe o número de vitórias (wins) e o número de empates (ties) e retorna a quantidade de pontos que o time marcou em um campeonato. Se ganhar vale 3 pts, se empatar ganha 1 pt.

Contar a quantidade

deverá retornar a quantidade de vezes que o maior número se repete ao receber uma string de números.

OS NÚMEROS DEVEM ESTAR SEPARADOS POR VÍRGULA. X, Y, Z. Não coloque os colchetes []

Calcular Áreas de Polígonos

Essa função calculará a área dos polígonos Triangulo ou quadrado

Caça ao Rato

Essa função verificará qual gato está mais perto do rato. Considere 1 para 1 unidade de distância

FizzBuzz [EM MANUTENÇÃO]

A função fizzBuzz recebe um array de números e para cada número do array é realizada a divisão por 3 e por 5 e de acordo com o resultado

  • Retorne a string 'fizz' para cada número do array que seja divisível apenas por 3;
  • Retorne a string 'buzz' para cada número do array que seja divisível apenas por 5;
  • Retorne a string 'fizzBuzz' para cada número do array que seja divisível por 3 e 5;
  • Retorne a string 'bug!' para cada número do array que não seja dividido por 3 nem por 5.

Codificador e Decodificador

Essa função codifica e decodifica uma frase, trocando vogais por números ou números por vogais.

Lista e objetos

Implemente a função techList que recebe um array e uma string e retorna um array de objetos.

Visite o arquivo JS para conferir a função ou use no console do navegador a função techList, adicione o primeiro argumento um array de alguma coisa, e um nome em seguida.


Tente usar: techList(['React', 'Jest', 'HTML', 'CSS', 'JavaScript'], 'Fernando');
