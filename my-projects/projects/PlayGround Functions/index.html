PlayGrounds Functions

Esses projetos utilizam essa interface gráfica para experimento das funções, o objetivo não foi estilizar com esmero a página. O objetivo é demosntrar as funções com suas funcionalidades.

Comparar dois elementos

Essa função pegará duas palavras e retornará se elas são iguais

Separa strings

Essa função pegará uma frase/tTexto e irá separar as palavras por vírgula

Concatenar Strings

Recebe uma string e deve retornar uma string com o formato `'ÚLTIMA PALAVRA, PRIMEIRA PALAVRA`, independente do tamanho do array.

Contar Pontos

Recebe o número de vitórias (wins) e o número de empates (ties) e retorna a quantidade de pontos que o time marcou em um campeonato. Se ganhar vale 3 pts, se empatar ganha 1 pt.

Contar a quantidade

deverá retornar a quantidade de vezes que o maior número se repete ao receber uma string de números.

OS NÚMEROS DEVEM ESTAR SEPARADOS POR VÍRGULA. X, Y, Z. Não coloque os colchetes []
